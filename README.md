# Suara.ai — Pelatih Public Speaking AI

Suara.ai adalah aplikasi web modern yang dirancang untuk membantu Anda meningkatkan keterampilan public speaking dengan bantuan kecerdasan buatan. Platform ini menyediakan coaching interaktif, analisis real-time, dan feedback personal untuk pengembangan kemampuan presentasi Anda.

## Fitur Utama

- 🎯 **Coaching Interaktif** - Mendapatkan bimbingan personal dari AI
- 📊 **Analisis Real-time** - Feedback mendalam tentang kualitas presentasi Anda
- 🎨 **Desain Modern** - Interface yang elegan dan mudah digunakan
- 📱 **Responsive Design** - Optimal di semua perangkat (desktop, tablet, mobile)
- 🌐 **Web-based** - Tidak perlu instalasi, akses langsung dari browser

## Teknologi yang Digunakan

- **Frontend:** HTML5, CSS3, JavaScript
- **Design System:** Custom CSS dengan typography premium
- **Deployment:** Vercel

## Instalasi Lokal

1. Clone repository ini:
```bash
git clone https://github.com/skilamardotilah/suara.ai.git
cd suara.ai
```

2. Buka file HTML di browser:
```bash
# Windows
start suara_ai.html

# macOS
open suara_ai.html

# Linux
xdg-open suara_ai.html
```

Atau gunakan live server:
```bash
npx http-server
```
